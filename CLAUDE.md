# CLAUDE.md

## 작업 개요

이 프로젝트는 React 기반의 웹 프론트엔드 어플리케이션입니다.
가독성, 컴포넌트의 재사용성, 그리고 일관된 UI 디자인을 최우선으로 합니다.

## 작업 환경

- React 19, TypeScript, Tailwind CSS v4, TanStack Query v5, React Router, @stomp/stompjs v7, Axios, Sonner

## 작업별 가이드라인 참조 명령 (중요)

1. UI 개발, 스타일 수정, 디자인 컴포넌트 작성 시:
   반드시 작업을 시작하기 전에 `.claude/DESIGN_SYSTEM.md` 파일을 읽고 해당 제약 조건을 파악하세요.
2. 코드 리팩토링, 컴포넌트 분리, 구조 개선, 비즈니스 로직 작성 시:
   반드시 작업을 시작하기 전에 `.claude/ARCHITECTURE.md` 파일을 읽고 아키텍처 규칙을 파악하세요.
3. 새로운 기능을 구현할 때 (UI + 로직 전체):
   `.claude/DESIGN_SYSTEM.md`, `.claude/ARCHITECTURE.md` 을 모두 읽고 결합하여 적용하세요.
