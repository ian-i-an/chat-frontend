# UI & 디자인 시스템 가이드

본 문서는 프로젝트의 시각적 일관성을 유지하기 위한 절대 규칙입니다. UI 변경 및 컴포넌트 추가 시 이 규칙을 엄격히 준수하세요.

## 스타일링 규칙

- 스타일링은 반드시 Tailwind CSS만 사용하며, CSS Module이나 인라인 스타일은 금지.
- 메인 색상은 `blue-500`사용. 디자인적 필요에 따라 이보다 밝은 단계(`blue-400`, `blue-300` 등)만 허용. `blue-600` 이상의 어두운 블루는 금지
- 텍스트 크기는 tailwind css의 text 규격을 따르지만(예, `text-sm`, `text-xl`),
  모바일 환경에서 입력 시 화면 자동 확장을 막기 위해 입력창엔 최소 `text-base` (16px) 사용.
